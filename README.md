# La Delice

### O melhor da gastronomia mundial.
<img src="https://github.com/scmfaria/la-delice/blob/master/icon.png" width="150" height="150"/>

O aplicativo La Delice conta com um estilo moderno e bem interativo.
Ele exibe as informações de forma rápida e simples, com uma interface de grande interação ao usuário.


## Tela Principal
A tela principal é composta pelo componente AppBarLayout. Para listar as opções de menu, foi utilizado o RecyclerView e foi implementado também um simples menu lateral, utilizando os componentes DrawerLayout e NavigationView.

<img src="https://github.com/scmfaria/la-delice/blob/master/2.png" width="220" height="400"/>
<img src="https://github.com/scmfaria/la-delice/blob/master/1.png" width="220" height="400"/>

## Tela de Detalhes do Menu
Ao clicar em um dos itens da listagem na tela principal, é aberto uma tela com o detalhamento deste menu, que contém uma lista de imagens, onde foi utilizado o componente CarouselView e logo abaixo uma descrição detalhada sobre este menu. Essas informações estão sob um CardView em uma tela com o estilo transparente, o que permite visualizar em volta do CardView a tela de Menu Principal.
